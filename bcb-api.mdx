## MCP Server

Conecta desde Claude, Cursor, VS Code o cualquier cliente MCP.

### Configuracion

```json
{
  "mcpServers": {
    "bcb": {
      "type": "http",
      "url": "https://bcb.cucu.bo/mcp"
    }
  }
}
```

### Tools

| Tool | Descripcion |
|------|-------------|
| `get_tc_referencial` | **TC referencial USD** â€” compra y venta. Usado para tarjetas de debito segun disposicion BCB. |
| `get_tc_oficial` | TC oficial USD/BOB (Bs 6.86 / 6.96). |
| `get_monedas` | **17 monedas internacionales** en Bs (EUR, BRL, PEN, ARS, GBP, JPY, CNY, etc.). |
| `get_ufv` | Unidad de Fomento a la Vivienda del dia. |
| `get_oro` | Oro y plata en USD por onza troy fina (O.T.F.). |
| `get_inflacion` | IPC, inflacion mensual y anual. |
| `get_tre` | Tasa de Referencia: MN, MVDOL, MN-UFV, ME. |
| `get_all_indicators` | Todos los indicadores en JSON. |

## REST API

Endpoints publicos. Sin autenticacion.

**Base URL:** `https://bcb.cucu.bo`

### TC Referencial USD

<CodeGroup>
```bash Request
curl https://bcb.cucu.bo/api/v1/tc/usd
```

```json Response
{
  "tc_referencial_usd": {
    "compra": 9.41,
    "venta": 9.61,
    "moneda": "USD/BOB",
    "fecha": "2026-04-09"
  },
  "fuente": "Banco Central de Bolivia (bcb.gob.bo)"
}
```
</CodeGroup>

### Todos los endpoints

| Metodo | Endpoint | Descripcion |
|--------|----------|-------------|
| `GET` | `/api/v1/tc/usd` | TC referencial USD (compra/venta) |
| `GET` | `/api/v1/tc/oficial` | TC oficial |
| `GET` | `/api/v1/tc/monedas` | 17 monedas internacionales en Bs |
| `GET` | `/api/v1/tc/ufv` | UFV del dia |
| `GET` | `/api/v1/tc/oro` | Oro y plata |
| `GET` | `/api/v1/tc/inflacion` | Indicadores de inflacion |
| `GET` | `/api/v1/tc` | Todos los indicadores |
| `GET` | `/health` | Health check |

### Ejemplo: TC oficial
