---

Documento de ajuste que permite corregir o anular parcialmente una factura ya emitida. Obligatorio para devoluciones despues de 30 dias.

## Datos del sector

| Campo | Valor |
|-------|-------|
| `codigoDocumentoSector` | `2` |
| `invoiceType` | `NOTE_CREDIT_DEBIT` |
| WSDL SIAT | `ServicioFacturacionDocumentoAjuste` |

## Campos adicionales

Ademas de los [campos estandar](/api-reference/create-invoice#request-body), este sector requiere:

### Cabecera

| Campo | Tipo | Req | Descripcion |
|-------|------|-----|-------------|
| `originalCuf` | string | Si | CUF de la factura original que se ajusta |
| `originalInvoiceNumber` | integer | Si | Numero de la factura original |
| `originalEmissionDate` | string | Si | Fecha de emision de la factura original (ISO 8601) |
| `amountTotalOriginal` | number | Si | Monto total de la factura original |
| `amountTotalReturn` | number | Si | Monto total devuelto al cliente |
| `amountDiscountCreditDebit` | number | Si | Monto de descuento aplicado en la nota |
| `amountCashCreditDebit` | number | Si | Monto efectivo de la nota credito/debito |

### Detalle

| Campo | Tipo | Req | Descripcion |
|-------|------|-----|-------------|
| `details[].codeDetailTransaction` | string | Si | Codigo de detalle de transaccion del SIN |
