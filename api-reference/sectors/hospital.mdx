Para hospitales, clinicas privadas y centros medicos con servicios especializados (cirugia, internacion, laboratorio hospitalario).

## Datos del sector

| Campo | Valor |
|-------|-------|
| `codigoDocumentoSector` | `15` |
| `invoiceType` electronica | `ELECTRONIC_HOSPITAL` |
| `invoiceType` computarizada | `COMPUTERIZED_HOSPITAL` |
| WSDL SIAT | `ServicioFacturacionElectronica` |

## Campos adicionales

Este sector requiere informacion detallada del servicio medico y profesional.

### Detalle

| Campo | Tipo | Req | Descripcion |
|-------|------|-----|-------------|
| `details[].specialty` | string | Si | Especialidad medica |
| `details[].specialtyDetail` | string | No | Detalle adicional de la especialidad |
| `details[].operatingRoomNumber` | string | No | Numero de quirofano o sala de operaciones |
| `details[].doctorSpecialty` | string | Si | Especialidad del medico tratante |
| `details[].doctorName` | string | Si | Nombre completo del medico |
| `details[].doctorDocument` | string | Si | NIT o documento del medico |
| `details[].doctorRegistrationNumber` | string | Si | Numero de matricula profesional |
| `details[].doctorInvoiceNumber` | string | No | Numero de factura del medico |
