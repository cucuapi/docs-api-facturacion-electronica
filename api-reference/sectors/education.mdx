Para colegios, universidades, institutos y centros de formacion.

## Datos del sector

| Campo | Valor |
|-------|-------|
| `codigoDocumentoSector` | `8` |
| `invoiceType` electronica | `ELECTRONIC_SECTOR_EDUCATION` |
| `invoiceType` computarizada | `COMPUTERIZED_SECTOR_EDUCATION` |
| WSDL SIAT | `ServicioFacturacionElectronica` |

## Campos adicionales

### Cabecera

| Campo | Tipo | Req | Descripcion |
|-------|------|-----|-------------|
| `studentName` | string | Si | Nombre completo del estudiante |
| `billedPeriod` | string | Si | Periodo facturado. Ej: `"Marzo 2026"`, `"1er Semestre 2026"` |
