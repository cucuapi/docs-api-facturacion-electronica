Para operaciones de exportacion de Bolivia al exterior. Tasa IVA cero.

## Datos del sector

| Campo | Valor |
|-------|-------|
| `codigoDocumentoSector` | `3` |
| `invoiceType` electronica | `ELECTRONIC_EXPORT` |
| `invoiceType` computarizada | `COMPUTERIZED_EXPORT` |
| WSDL SIAT | `ServicioFacturacionCompraVenta` |

## Campos adicionales

### Detalle

| Campo | Tipo | Req | Descripcion |
|-------|------|-----|-------------|
| `details[].codeNandina` | string | Si | Codigo arancelario Nandina del producto exportado |
