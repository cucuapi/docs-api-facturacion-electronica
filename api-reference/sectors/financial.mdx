Para bancos, cooperativas de ahorro, mutuales, fondos financieros y entidades reguladas por ASFI.

## Datos del sector

| Campo | Valor |
|-------|-------|
| `codigoDocumentoSector` | `23` |
| `invoiceType` | `ELECTRONIC_FINANCIAL` |
| WSDL SIAT | `ServicioFacturacionEntidadFinanciera` |

## Campos adicionales

Este sector no requiere campos adicionales mas alla de los [campos estandar](/api-reference/create-invoice#request-body). Los datos financieros especificos (numero de cuenta, tipo de operacion) se incluyen en la descripcion de los items.
