El documento sector mas comun. Cubre la venta de bienes y servicios generales.

## Datos del sector

| Campo | Valor |
|-------|-------|
| `codigoDocumentoSector` | `1` |
| `invoiceType` electronica | `ELECTRONIC_SALE` |
| `invoiceType` computarizada | `COMPUTERIZED_SALE` |
| WSDL SIAT | `ServicioFacturacionCompraVenta` |

## Campos adicionales

Este sector no requiere campos adicionales mas alla de los [campos estandar](/api-reference/create-invoice#request-body).

Los campos `serialNumber` e `imeiNumber` en los items del detalle son opcionales y se usan cuando se facturan equipos con numero de serie o IMEI.
