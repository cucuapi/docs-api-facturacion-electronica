Para aerolineas y empresas de transporte aereo comercial.

## Datos del sector

| Campo | Valor |
|-------|-------|
| `codigoDocumentoSector` | `13` |
| `invoiceType` | `ELECTRONIC_AIRLINE` |
| WSDL SIAT | `ServicioFacturacionElectronica` |

## Campos adicionales

Este sector no requiere campos adicionales mas alla de los [campos estandar](/api-reference/create-invoice#request-body). La informacion del vuelo se incluye en la descripcion de los items del detalle.
