Para operaciones con tasa IVA cero: exportacion de servicios, diplomticos, organismos internacionales y otros casos especificos definidos por el SIN.

## Datos del sector

| Campo | Valor |
|-------|-------|
| `codigoDocumentoSector` | `5` |
| `invoiceType` | `ELECTRONIC_DUTY_FREE` |
| WSDL SIAT | `ServicioFacturacionCompraVenta` |

## Campos adicionales

Este sector no requiere campos adicionales mas alla de los [campos estandar](/api-reference/create-invoice#request-body).

<Warning>
  El tipo de documento del cliente (`clientDocumentType`) para tasa cero debe ser `5` (NIT) y el numero de documento NO puede ser `0`. Use un numero valido como `99001`.
</Warning>
