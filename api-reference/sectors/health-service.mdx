Para consultorios medicos, laboratorios clinicos, farmacias y servicios de salud en general.

## Datos del sector

| Campo | Valor |
|-------|-------|
| `codigoDocumentoSector` | `10` |
| `invoiceType` | `ELECTRONIC_HEALTH_SERVICE` |
| WSDL SIAT | `ServicioFacturacionElectronica` |

## Campos adicionales

Este sector no requiere campos adicionales mas alla de los [campos estandar](/api-reference/create-invoice#request-body).
