## Preguntas frecuentes

<AccordionGroup>
  <Accordion title="Como empiezo a integrar?">
    Contrata tu plan y recibiras acceso al **Sandbox** (SIAT Piloto) donde puedes desarrollar y validar tu integracion completa. Con el **MCP Server** incluido, tu AI (Claude, Cursor, VS Code) se conecta directamente — pasale tu API Key y empieza a facturar con lenguaje natural.
  </Accordion>

  <Accordion title="Que incluye el pago inicial de Bs. 1,800?">
    El pago inicial unico cubre: configuracion completa de tu cuenta ante el SIAT (Impuestos Nacionales), soporte durante el proceso de certificacion, integracion de tu firma digital, y onboarding tecnico con tu equipo. Es un pago unico, no recurrente.
  </Accordion>

  <Accordion title="Cual es la diferencia entre los planes?">
    Todos los planes incluyen la misma funcionalidad core: emision, anulacion, PDF, email, API, MCP Server y AI Assistant. La diferencia esta en las personalizaciones: **Semi-Senior** agrega plantilla PDF personalizada y DNS propio; **Senior** agrega ademas soporte prioritario. El costo por factura tambien baja con cada plan superior.
  </Accordion>

  <Accordion title="Puedo cambiar de plan?">
    Si, puedes subir de plan en cualquier momento. El cambio se aplica inmediatamente. Contacta a ventas@cucu.bo para gestionar el cambio.
  </Accordion>

  <Accordion title="Ofrecen facturacion anual?">
    Si. Ofrecemos facturacion mensual y anual. Los planes anuales incluyen descuento. Contacta a ventas@cucu.bo para cotizacion anual personalizada.
  </Accordion>

  <Accordion title="Que metodos de pago aceptan?">
    Transferencia bancaria, QR Bolivia y tarjeta de credito/debito. Emitimos factura por cada pago.
  </Accordion>

  <Accordion title="Necesito certificado digital para el sandbox?">
    No. El sandbox no requiere certificado digital, NIT real ni ningun tramite ante Impuestos Nacionales. Es un ambiente completo para desarrollo y pruebas.
  </Accordion>

  <Accordion title="El MCP Server esta incluido en todos los planes?">
    Si. El MCP Server con las 14 herramientas esta incluido en todos los planes. Conecta Claude, Cursor, VS Code o cualquier cliente MCP compatible. Pasale tu API Key a tu AI y empieza a facturar con lenguaje natural.
  </Accordion>

  <Accordion title="El AI Assistant esta incluido en todos los planes?">
    Si. El AI Assistant con RAG sobre normativa SIAT boliviana esta disponible en todos los planes. Te ayuda con dudas sobre facturacion, codigos de error del SIN, y buenas practicas.
  </Accordion>

  <Accordion title="Que pasa si necesito mas de un cluster?">
    Contacta a ventas@cucu.bo para planes multi-cluster o enterprise personalizados.
  </Accordion>
</AccordionGroup>
